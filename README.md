# Somaliunfiedpaymentsystrm

The UnifiedPay Somalia Platform is a next-generation fintech infrastructure designed to solve a critical structural inefficiency in Somalia’s financial ecosystem — the lack of interoperability between mobile money providers and banks.

## Quick start

### Run the demo

```bash
python3 app.py
```

Expected output (example):

```text
SUCCESS: Sent $25.00 from Asha to Hassan via interoperability switch (ZAAD -> UnifiedPay -> Premier Bank).
```

### Run tests

```bash
python3 -m unittest -v
```
